# 计算出iphone和pc&laptop中text渲染时的实际像素值的公式

### iphone 7 plus's screen

-  1rem = 16px
-  4rem = 16px * 4 = 64px

###  三星23.6英寸 LED电脑显示器

- 1rem = 16px
- 2rem = 16px * 2 = 32px
